# EDA-Ecommerce-Analysis
📊 Exploratory Data Analysis — E-commerce Dataset
Tool Used: Microsoft Excel
Dataset: 1,200 rows | 14 columns
Internship: DecodeLabs Data Analytics Internship | May 2026

📌 Project Goal
To analyze a real-world e-commerce dataset and uncover patterns, trends, and distributions through descriptive statistics, visual charts, and data-driven observations.

📌 What This Project Covers
✅ Descriptive Statistics (13 measures across Total Price, Unit Price and Quantity)
✅ Strategic Performance Trends by Product Category
✅ Outlier Detection using the IQR Method
✅ Customer & Referral Channel Performance
✅ Discount Code Performance Analysis

📌 Charts Created
📊 Total Revenue by Product Category — Bar chart comparing revenue across all 7 product categories
📊 Revenue by Discount Code — Bar chart showing revenue performance across all 4 coupon codes
📊 Orders by Referral Source — Chart comparing customer acquisition across Instagram, Facebook, Email, Referral and Google

📌 Key Observations

The average order value was $1,053.97 but the median of $823.62 reveals that high-value orders were pulling the average upward
Printer and Chair were the top revenue generating categories at approximately $195K each while Phone was the lowest at $151,722.39
8 outlier orders were statistically identified above the IQR threshold of $3,330.41 — all had a quantity of 5 items suggesting bulk purchases drive extreme values
Instagram was the strongest customer acquisition channel with 259 orders and $275,285.45 in revenue while Referral had the lowest performance
FREESHIP was the most popular coupon code driving 313 orders and $335,036.99 in revenue — yet average order values remained consistent across all codes suggesting discounts influenced choice but not spending size
Cart size was consistent across all referral channels averaging between 5.3 and 5.6 items showing uniform buying behavior regardless of how customers found the store
